~~~~~妊娠期危险度分级和哺乳期用药安全性（5）*****

表8A（续）FDA妊娠危险度分级和哺乳期用药安全性

部分药物具有多种抗菌活性，如阿米卡星、氨苯砜。由于篇幅有限，每种药物仅列出一项主要类别，如利福平归类在抗分枝杆菌类别。

类别/药物 FDA
妊娠危险度		 哺乳期用药
青霉素V钾 B 乳汁浓度低，不良反应可能性极低，监测婴儿胃肠道毒性
匹美西林 B 乳汁浓度低，不良反应可能性极低，监测婴儿胃肠道毒性
普拉佐米星 C 可能安全，监测婴儿胃肠道毒性
多黏菌素B C 局部应用安全（无全身用药数据）
奎奴普丁-达福普汀 B 仔细监测，可能安全，但尚无数据；如果可能，避免使用
瑞他莫林 B 安全性尚未确定，避免使用
利福霉素SV C 无数据，可能安全（由于口服给药后的全身吸收可忽略不计）
利福昔明 C 仔细监测，可能安全，但尚无数据；如果可能，避免使用
沙瑞环素 — 勿用（胎儿损害风险、牙齿变色、骨骼生长抑制）
泰地唑胺 C 人类数据不充分。在大鼠和小鼠有胚胎毒性
舒巴坦+度洛巴坦 — 舒巴坦：人类或动物无毒性证据。度洛巴坦：无人类数据，小鼠骨骼变异的证据
硫培南酯-丙磺舒 C 硫培南：人类无毒性证据，动物有母体毒性证据，对乳汁分泌的影响尚不明确，对母乳喂养婴儿的影响尚不明确。丙磺舒：可分泌至乳汁，目前未报告对母乳喂养婴儿的不良影响
替莫西林 B 人类或动物无毒性证据
特拉万星 C 基于动物实验研究，存在胚胎毒性风险。勿用，除非效益超过胚胎风险
泰利霉素 C 仔细监测，可能安全，但尚无数据；如果可能，避免使用

注：大写字母（A、B、C、D、X）=旧的FDA风险分类，该列文字=新的FDA风险分类。
