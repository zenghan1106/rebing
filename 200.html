~~~~~妊娠期危险度分级和哺乳期用药安全性（9）*****

表8A（续）FDA妊娠危险度分级和哺乳期用药安全性

部分药物具有多种抗菌活性，如阿米卡星、氨苯砜。由于篇幅有限，每种药物仅列出一项主要类别，如利福平归类在抗分枝杆菌类别。

类别/药物 FDA
妊娠危险度		 哺乳期用药
抗寄生虫药物（续）
蒿甲醚-本芴醇 — 根据WHO和CDC的证据审查和政策，可在妊娠全期使用
五价锑 — 无数据，但很可能有胚胎毒性。孕期禁用
青蒿琥酯 — 根据WHO和CDC的证据审查和政策，可在妊娠全期使用
阿托伐醌 C 人类数据不充分，动物无毒性证据
阿托伐醌-氯胍 C 可能安全，尚未修改说明书
苯硝唑 — 动物实验提示存在胎儿风险。建议对具有生育能力的妇女进行妊娠检测
磷酸氯喹 C 可能安全，监测，但数据有限；如果可能，勿用
乙胺嗪 C 妊娠期禁用
双氢青蒿素-哌喹 — 人类数据不充分，勿用
呋喃二氯尼特 — 人类无数据，动物无毒性证据
依氟鸟氨酸 — 禁忌使用，但作为非洲锥虫病的抢救性治疗时可使用
非昔硝唑 — 动物实验显示超常剂量对大鼠和兔具有毒性，但冈比亚布氏锥虫感染患者可在妊娠中晚期用
Fumagillin — 无数据

注：大写字母（A、B、C、D、X）=旧的FDA风险分类，该列文字=新的FDA风险分类。
